# Riolabz Website UI/UX Redesign

A modern UI/UX redesign of the Riolabz website created as part of a UI/UX design assignment. The project focuses on a clean, responsive, and user-friendly interface while maintaining a professional IT/Technology brand identity.

---

## 📌 Project Overview

The objective of this project was to redesign the existing Riolabz website with a completely fresh design approach instead of copying the current website.

The redesign emphasizes:

- Modern UI/UX principles
- Improved user experience
- Responsive layouts
- Professional IT company branding
- Clean typography and spacing
- Attractive color palette
- Smooth animations and interactions
